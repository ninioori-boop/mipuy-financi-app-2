## 🔌 מתגי קונסולה שנשארו (אצלך, לא דחוף)
1. **הגג היומי** — `AI_DAILY_LIMIT=<מספר>` ב‑Vercel (כרגע כבוי).
2. **ניקוי אוטומטי (TTL)** על הקולקציה `rateLimits`, שדה `expireAt` (Firestore → TTL).
3. **App Check** — רישום reCAPTCHA v3 → `NEXT_PUBLIC_RECAPTCHA_SITE_KEY` → `APP_CHECK_ENFORCE=true`. ⚠️ **רק על ה‑routes, לא על Firestore** (אכיפה על Firestore תשבור את האפליקציה הישנה).

## 🚨 כפתור חירום (לשעת התקפה בלבד)
ב‑Vercel: `AI_KILL_SWITCH=true` → כל ה‑AI נעצר תוך שנייה. למחוק את המשתנה כדי להחזיר. (מתועד ב‑[incident-runbook.md](incident-runbook.md) §1.)
